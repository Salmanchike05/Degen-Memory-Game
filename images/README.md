# Папка для иконок токенов

Поместите в эту папку изображения токенов с следующими именами:

**Основные токены (обязательно):**
- `brett.png` - иконка токена BRETT
- `toshi.png` - иконка токена TOSHI  
- `degen.png` - иконка токена DEGEN

**Дополнительные токены (для 10 токенов):**
- `bitcon.png` - иконка токена bitcoin
- `solana.png` - иконка токена solana
- `toby.png` - иконка токена toby
- `base.png` - иконка токена BASE
- `ponke.png` - иконка токена ponke
- `usdc.png` - иконка токена USDC
- `ethereum.png` - иконка токена ethereum

**Рекомендации:**
- Формат: PNG или JPG
- Размер: рекомендуется квадратные изображения (например, 256x256px или 512x512px)
- Прозрачный фон предпочтителен для лучшего отображения

Вы можете найти иконки токенов Base на:
- CoinGecko: https://www.coingecko.com
- Официальных сайтах проектов
- DEXScreener или других криптографических платформах
