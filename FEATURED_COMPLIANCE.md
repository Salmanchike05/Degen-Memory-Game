## ❌ Требует исправления

### 1. Authentication ❌
**Требование:** Автоматическое подключение кошелька через Base Account, без внешних редиректов

**Проблема:** 
- Используется `window.ethereum` (MetaMask) напрямую
- Нет интеграции с Base Account SDK
- Возможны внешние редиректы

**Нужно:**
- Использовать `sdk.context.user` из MiniApp SDK для получения информации о пользователе
- Убрать прямую работу с MetaMask
- Использовать Base Account для автоматической аутентификации

### 2. Onboarding Flow ❌
**Требование:** Понятные инструкции при старте, отображение аватара и username (не 0x адреса)

**Проблема:**
- Нет онбординга/инструкций при первом запуске
- Показываются 0x адреса в таблице лидеров
- Нет отображения аватара/username пользователя

**Нужно:**
- Добавить онбординг с инструкциями игры
- Использовать `sdk.context.user` для получения username/аватара
- В таблице лидеров показывать username вместо адресов

### 3. Base Compatibility ⚠️
**Требование:** Клиент-агностичность, спонсируемые транзакции

**Проблема:**
- Текущий код не использует транзакции, но при добавлении нужно будет спонсировать их

**Нужно:**
- Убедиться, что нет упоминаний Farcaster или других клиентов
- При добавлении транзакций использовать спонсирование

### 6. Usability ❌
**Требование:** Поддержка светлой и темной темы, минимум 44px для touch targets

**Проблема:**
- Только темная тема
- Размеры кнопок могут быть меньше 44px

**Нужно:**
- Добавить поддержку светлой темы через `prefers-color-scheme`
- Убедиться, что все кнопки минимум 44x44px

### 7. App Metadata ❌
**Требование:**
- Иконка: 1024×1024px, PNG, без прозрачности
- Обложка: 1200×630px (1.91:1), высокое качество
- 3 скриншота: 1284×2778px (portrait orientation)

**Проблема:**
- Текущие иконки 100x100px или меньше
- Нет обложки нужного размера
- Нет скриншотов нужного размера

**Нужно:**
- Создать иконку 1024×1024px без прозрачности
- Создать обложку 1200×630px
- Создать 3 скриншота 1284×2778px
